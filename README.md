# Simple Drawing App

- p5js로 만든 간단한 그림판 앱

## 목적

- p5js DOM 연습
- 드로잉 툴 원리 익히기

## 기능
- 자유선 (색상, 굵기 선택)
- 직선 (색상, 굵기 선택)
- 지우개 (굵기 선택)
- 스프레이 (색상 선택)
- 모두 지우기
- png 파일로 저장
- 다른 도구를 선택해도 색상, 굵기 등 옵션 상태가 유지될 수 있도록 함
